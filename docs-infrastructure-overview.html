Introduction

This section of the documentation decribes how the MetaMechanists infrastructure is set up at a high level. First, it is important to know:

  • Our infrastructure is hosted by Void (free of charge) but Void doesn't have much involvement in it, so administration is done remotely unless in exceptional circumstances
  • We generally access the machine via SSH, ask Idra for details on how to do this + set up SSH keys
  • Use VSCode to edit and upload/download files, ask Idra for details on how to do this
  • Some files are permission restricted and cannot be edited from VSCode. Ask Idra on how to deal with this
