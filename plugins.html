PLUGINS
Standalone Plugins
  • ArtMap - create pixel art in-game
  • AureliumSkills - MCMMO-like plugin which adds skills and abilities
  • BungeeTabListPlus - handles the fancy tab list
  • CMI - large general-purpose plugin adding teleports, homes, QoL commands, and much more
  • CMILib - dependency of CMI
  • Craftbook - adds lots of new immersive mechanics
  • CraftingStore - handles donations
  • ContainerPassthrough - allows you to access containers through item frames
  • CoreProtect - logging plugin, primarily used to revert griefs
  • DiscordSRV - handles Discord chat, console, and rank syncing
  • ExcellentCrates - adds loot crates and crate keys
  • FastAsyncWorldEdit - dependency of Craftbook and used for staff building projects
  • FastChunkPregenerator - pregenerates chunks to reduce server load
  • GriefPrevention - land claiming plugin
  • GPPressurePlates - tiny custom plugin that fixes GP pressure plate compatiblity
  • Harbor - highly configurable sleeping plugin
  • InteractiveChat - allows players to send interactive chat messages such as showing your inventory
  • InteractiveChatDiscordSrvAddon - provides Discord hooks for InteractiveChat
  • LuckPerms - permission management plugin with a web interface
  • MetaRank - custom ranking plugin specifically designed for Slimefun centric ranks
  • NerfPhantoms - allows phantoms to be toggled for each player
  • NexEngine - dependency of ExcellentCrates
  • OtherAnimalTeleport - makes nearby tamed animals teleport with the player
  • PlaceholderAPI - allows other plugins to use placeholders such as number of online players
  • Plan - player analytics plugin
  • ProtocolLib - dependency of several plugins
  • PvPManager - allows for toggling PvP
  • ResourceWorld - handles resource worlds to prevent resource depletion
  • Slimefun - adds hundreds of new items
  • SurvivalInvisItemFrames - adds craftable invisible item frames
  • StaffPlusPlus - staff plugin providing bans, warns, staffchat, and more
  • StaffPlusPlusDiscord - provides Discord hooks for StaffPlusPlus
  • Spark - performance analysis plugin
  • Squaremap - renders 2D map which can be accessed here
  • Vault - dependency of a lot of plugins
  • VotingPluging - manages rewards for voting

Slimefun Addons
  • CrystamaeHistoria - adds an extensive magic plugin with spells and a progression system
  • DankTech - adds backpacks with functionality such as auto-deleting specific items
  • DyedBackpacks - adds dyed variants of Slimefun backpacks
  • EcoPower - adds new ways to renewably generate power
  • ExoticGarden - adds new plants and foods
  • ExtraTools - adds more tools that can be made from Slimefun materials
  • FluffyMachines - adds quality-of-life machines and items
  • FoxyMachines - adds miscallenous magic and technology items and machines
  • Galacitfun - adds rockets, stargates, new planets, and other space stuff
  • InfinityExpansion - massive end game expansion adding ridiculously powerful items and machines
  • Miniblocks - adds miniature versions of certain blocks for building
  • Netheopoiseis - allows the Nether to be terraformed
  • Networks - adds a digital item storage system, compatible with Infinity Expansion storage units
  • RelicsOfCthonia - adds relics that randomly drop and can be traded with Piglins for Slimefun items
  • SFCalc - provides commands to calculate the raw resources needed to make a Slimefun item
  • SlimeHUD - adds a HUD to tell you what Slimefun block you're looking at
  • SlimeTinker - allows you to create hugely customisable tools
  • SlimyTreeTaps - adds a renewable source of plastic
  • SoulJars - adds items that can be used to change the type of spawners
  • Transcendence - allows you to gain powerful semi-permanent status effects
  • Wildernether - adds Nether-specific tools, armour, and other items
  • SFMM - our own custom plugin with lots of QoL stuff and new machines
