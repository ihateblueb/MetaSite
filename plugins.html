Plugins
Plugins running on the survival server only. Last updated 21/12/2021

Standalone Plugins
  • CMI - large general-purpose plugin adding teleports, homes, QoL commands, and much more
  • CMILib - Dependency of CMI
  • CoreProtect - logging plugin, primarily used to revert griefs
  • DiscordSRV - handles Discord chat, console, and rank syncing
  • GriefPrevention - land claiming plugin
  • Harbor - highly configurable sleeping plugin
  • InteractiveChat - adds interactive messages such as [inv]
  • LuckPerms - permission management plugin with a web interface
  • McMMO - adds skills that you can level up to gain advantages like more damage in combat
  • MetaRank - custom ranking plugin specifically designed for Slimefun centric ranks
  • PlaceholderAPI - allows other plugins to use placeholders such as number of online players
  • Plan - player analytics plugin
  • PvPManager - allows for toggling PvP
  • Slimefun - adds hundreds of new items
  • Spark - performance analysis plugin
  • TABBukkitBridge - bridges TAB plugin between the server and BungeeCord
  • Vault - dependency of a lot of plugins
  • ViaBackwards - allows clients to connect using version 1.12+
  • ViaVersion - dependency of ViaBackwards
  • VotingPluging - manages rewards for voting
Slimefun Addons
  • DankTech - adds backpacks with functionality such as auto-deleting specific items
  • DyedBackpacks - adds dyed variants of Slimefun backpacks
  • EcoPower - adds new ways to renewably generate power
  • ExoticGarden - adds new plants and foods
  • ExtraHeads - adds heads that randomly drop from mobs
  • ExtraTools - adds more tools that can be made from Slimefun materials
  • FluffyMachines - adds quality-of-life machines and items
  • FoxyyMachines - adds miscallenous magic and technology items and machines
  • Galacitfun - adds rockets, stargates, new planets, and other space stuff
  • HotbarPets - adds pets that can be crafted and give you special abilities
  • InfinityExpansion - massive end game expansion adding ridiculously powerful items and machines
  • Networks - adds a digital item storage system, compatible with Infinity Expansion storage units
  • SlimeCustomizer - plugin for creating custom machines
  • SlimeTinker - allows you to create hugely customisable tools
  • SlimyTreeTaps - adds a renewable source of plastic
  • SoulJars - adds items that can be used to change the type of spawners
