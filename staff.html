Staff Tags

Owner

I (Idra) manage the staff team and do the large majority of behind-the-scenes maintenance.


Host

The Host (Void) kindly provides all our server hardware, free of charge. He does not have much involvement in the day-to-day running of the server.


Event Organiser

Event Organisers design events and handle event logistics.


Head builder

The Head Builder (Aria_Night) manages Builders and plans building projects.


Admin

Admins handle technical details of the server. Contact an Admin if you have a suggestion, find an issue, or lose items to a bug.


Game Moderator

Game Moderators deal with griefs and hacking. They have permission to use all coreprotect commands and edit inventories. Contact a Game Moderator if a player is causing issues in-game such as griefing.


Chat Moderator

Chat Moderators keep chat civil. Contact a Chat Moderator if you experience issues with another player in chat, both public and private.


Builder

Builders design and build spawns and minigames under supervision of the Head Builder.
